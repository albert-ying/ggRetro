`ggRetro`
=========

`ggRetro` is a package for creating floating axes, like in base R.

Basic Usage
-----------

    library(dplyr)
    library(ggplot2)
    library(ggRetro)

    p = ggplot(mtcars) +
      geom_line(aes(mpg, wt)) +
      theme_bw()
    p |> base_mode()

    ## [1] "Both numeric"

    p |> base_facet(c("vs"))
